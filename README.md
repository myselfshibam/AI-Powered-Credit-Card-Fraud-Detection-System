# 💳 AI-Powered Credit Card Fraud Detection System

An end-to-end Machine Learning project that detects fraudulent credit card transactions using advanced ML algorithms and an interactive Streamlit dashboard. This system analyzes transaction patterns to identify suspicious activities with high accuracy on an imbalanced financial dataset.

## 🚀 Features

* Fraud detection using Machine Learning
* Interactive Streamlit dashboard
* Real-time CSV upload support
* Data visualization with Plotly
* ROC-AUC & performance analytics
* Corporate-style responsive UI

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas & NumPy
* Scikit-learn

## 📊 ML Workflow

Data preprocessing → Feature engineering → Model training → Evaluation → Deployment

## ▶️ Run Project


pip install -r requirements.txt
streamlit run app.py


## 📈 Results

Achieved high fraud detection performance using classification algorithms and visualization-based insights for transaction monitoring.
